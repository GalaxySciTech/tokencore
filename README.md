# tokencore

#### 这只是一个数字货币的钱包后台组件

## 但是我在它的上层做了一件更大的事情：
 #### 我通过它构建了一个支持充值 提现 归集 为一体的区块链钱包后台

- #### 该钱包后台提供一个即插即用的区块链组件，可以随时获取BTC,ETH,USDT等区块链地址，并且内置归集功能，开放提现接口


#### 全套系统架构情况：
- 使用springboot框架
- 语言为java,kotlin
- 使用rabbitmq消息队列
- mysql云数据库
- xxl-job分布式定时任务框架

#### 全套系统均由我一个人开发，并且已经在生产环境跑了上亿流水，具有充足的可靠性，可扩展性，实用性，可以放心使用。
