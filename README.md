# tokencore

#### 这只是一个数字货币的钱包后台组件

## 但是我在它的上层做了一件更大的事情：
 #### 我通过它构建了一个支持充值 提现 归集 为一体的区块链钱包后台

- #### 该钱包后台提供一个即插即用的区块链组件，可以随时获取区块链地址，支持(BTC,,OMNI,ETH,ERC20,TRX,TRC20)的充归提


#### 全套系统架构情况：
- 使用springboot框架
- 语言为java,kotlin
- 使用rabbitmq消息队列
- mysql云数据库
- xxl-job分布式定时任务框架
